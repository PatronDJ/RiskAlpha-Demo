Investor reporting engine
Automated NAV, financial statements and investor deliverables — generated directly from the data warehouse. Zero manual assembly.
Inputs — from loan engine + fund register (via data warehouse)
Input 1
Loan positions
Daily balances · accruals · cashflow actuals from loan_positions_daily
Input 2
Waterfall output
Per-investor distribution entitlements from waterfall_distributions
Input 3
Fund register
Investor units, capital contributions and redemptions from dim_investor
Input 4
Valuations
Asset fair values from external appraisals — drives NAV and LVR calculations
all inputs flow through dbt mart layer in Snowflake
Reporting calculations — period-based financial computations
Calc 1
NAV calculation
Gross asset value less liabilities divided by units on issue = NAV per unit
Calc 2
Return metrics
IRR (XIRR) · MOIC · DPI · RVPI · TWR · MWR — since inception and period
Calc 3
Financial statements
P&L, balance sheet and cash flow — fund level per reporting period
Calc 4
Distribution calc
Income entitlement per investor unit · tax components · reinvestment option
period lock · 4-eyes approval · immutable snapshot
Period close — lock, review and approve before any statement is generated
1
Period lock
Month/quarter-end date reached. Calculation engine writes final snapshot to Snowflake with is_period_end = true. No further edits to source data.
2
Automated reconciliation
dbt tests run automatically — bank reconciliation, loan balance check, total assets = total liabilities + net assets. Any failure blocks report generation.
3
4-eyes approval
Fund manager reviews NAV calculation in approval portal. Signs off — approved_by and approved_at written to fund_nav. is_locked set to true. Audit trail complete.
approved data → document generation pipeline
Document generation — approved data → branded investor deliverables
Output 1
Investor statement (PDF)
Holdings, returns, distributions, fees — per investor. Generated by Templafy or Python/WeasyPrint. Stored in Azure Blob.
Output 2
Quarterly fund report
Portfolio narrative, asset-level commentary, covenant status table, performance attribution
Output 3
Distribution notice
Per-investor distribution amounts, tax breakdown (AMMA), payment instruction
